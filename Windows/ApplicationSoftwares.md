## Installation of Packages on R on Windows

```
install.packages(c("vegan", "colorspace", "RColorBrewer", "tidyverse", "ggsci", "khroma", "picante", "bipartite", "geosphere", "mpmcorrelogram", "pvclust", "foreach", "doParallel"))
```
